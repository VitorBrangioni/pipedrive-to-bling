# Pipedrive to Bling

Transfer pipedrive deals to bling like a sales order

## Config. API Token
`/src/config/apiTokens.js`


## Start API

1. `npm install`
2. `docker-compose build`
3. `docker-compose up api`

API port: 3000

## Production Environment

ip/port: `52.67.94.106:3000`

Host: AWS EC2
